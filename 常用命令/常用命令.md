## windows

### ssh

Windows查看ssh公钥方法

    1.打开你的git bash 窗口
    2.进入.ssh目录： cd ~/.ssh
    3.找到id_rsa.pub文件： ls
    4.查看公钥： cat id_rsa.pub 或者 vim id_rsa.pub
    5.绝对路径cat ~/.ssh/id_rsa.pub
taskkill /im 进程名 /f  杀死应用进程
